# HR Training Games App

## Overview
This application is designed to enhance HR training through interactive games. It aims to improve employee engagement, learning retention, and skill development in a fun and effective way.

## Features
- Multiple training game modules (quizzes, simulations, role-play)
- Progress tracking and analytics
- Leaderboards and rewards
- Customizable content for different HR topics
- User-friendly interface
